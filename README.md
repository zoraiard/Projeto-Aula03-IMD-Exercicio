## Aula 03 - Validação de Campos no JSF


##  Passo 1 – Criação dos modelos
Crie uma classe para representar um veículo. Um veículo deve possuir os seguintes atributos:
* Placa
* Ano
* Modelo
* Chassi

Crie uma classe para representar um motorista. Um motorista deve possuir os seguintes atributos:

* Nome
* Endereço
* Veículo

Crie uma classe para representar um ponto de taxi. Um ponto de taxi deve possuir os seguintes atributos:
* Endereço
* Horário Início Funcionamento
* Horário Fim Funcionamento
* Motorista Responsável
